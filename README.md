# 🚆 Railway Ticket Management System

A Java Full Stack web application that allows users to search trains, book railway tickets, cancel bookings, and manage reservations. The application provides a simple and user-friendly interface for passengers and an admin panel for managing train details.

---

## 📌 Features

- User Registration & Login
- Secure Authentication
- Book Railway Tickets
- View Booking History
- Admin Dashboard
- Manage Trains and Users
- Responsive User Interface
